# О чём этот проект
Автоматическая проверка доступности сайтов - если код ответа от главной страницы сайта НЕ 200 - пользователь получает сообщение в Telegram об изменении статуса с некоторой информацией:
1. Новый статус ответа сайта
2. Описание полученного статуса
3. Время последнего успешного/ошибочного статуса.

Также доступны различные команды администратора бота;

## Запуск проекта
Проект можно запустить используя Docker файл при условии уже работающего postgresql.

## Наполнение .env файла.
Файл ожидается в корне проекта. Ему требуются следующие данные:
  DB_HOST: str
  DB_PORT: int
  DB_USER: str
  DB_PASS: str
  DB_NAME: str
  BOT_TOKEN: str
