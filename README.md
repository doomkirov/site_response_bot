# О чём этот проект
Автоматическая проверка доступности сайтов - если код ответа от главной страницы сайта НЕ 200 - пользователь получает сообщение в Telegram об изменении статуса с некоторой информацией:
1. Новый статус ответа сайта
2. Описание полученного статуса
3. Время последнего успешного/ошибочного статуса.
